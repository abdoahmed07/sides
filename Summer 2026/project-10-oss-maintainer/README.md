understanding of the tool from the user side to contribute meaningfully from the code side.

## The key insight from this project

The gap between contributor and recurring contributor is mostly just showing up consistently
and being responsive. Three merged PRs over 8 weeks isn't a lot of code — but it's
enough to be a recognisable name. The maintainers started responding to my comments
faster because they knew from history that my observations were usually accurate.

That recognition is the actual goal. It means when I reach out for a reference or mention
this project in an interview, there's a real person who knows my work.
